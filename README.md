# Covenant Landing Page

A minimalist landing page with a clean structure.

## Structure

```
.
├── index.html           # Main landing page
├── favicon/             # Favicon assets (SVG, ICO, PNG formats)
└── station.JPG          # Background image
```

## Local Development

Start a local server:
```bash
python3 -m http.server 8000
```

Visit `http://localhost:8000`

## Features

- Responsive design
- Multi-format favicon support
- Single HTML file with embedded styles
